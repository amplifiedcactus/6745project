To compile aagtopoly.cpp:
g++ aagtopoly.cpp -o aagtopoly

To verify a multiplier with aagtopoly:
Convert multiplier to .aag filetype using aiger library
./aagtopoly [input_filename].aag [output_filename].sing 
Singular < "[output_filename].sing";

Examples provided:
mult.blif is a 2x2 bit multiplier
mult.aag is the .aag file generated by aiger library for mult.blif
multout.sing is the output from running aagtopoly on mult.aag

Remember that inputs and outputs in blif file or .aag file need to be in correct order for it to work:
In blif it should be:
.inputs a0 a1 [... ai] b0 b1 [... bi]
.outputs z0 z1 z2 z3 [...zi]

In aag it should be:
a0
a1
...
ai
b0
b1
...
bi
z0
z1
...
zi
